# selbynet
Social Network // SUNY Poly Capstone Spring 2021

Selbynet was developed as part of a capstone project at SUNY Polytechnic University,
in Spring of 2021, under the supervision of Scott Spetka.

Tools Used:
- NodeJs
- Socket.io
- MongoDB, Mongo Atlas
- mongoose
- WebRTC
- EJS [template engine]
